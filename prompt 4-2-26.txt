# 📋 ESPECIFICACIONES GENERALES

## Descripción del Proyecto

Desarrollar una **plataforma web progresiva (PWA)** de bienestar mental y neuronal que combine:
- Detección biométrica (PPG vía cámara + análisis de audio)
- IA generativa para experiencias personalizadas
- Sincronización grupal en tiempo real
- Módulos especializados: Meditación, Estrés, Sueño, Rendimiento, Sincronización Grupal
- Sistema de monetización completo
- Biblioteca de contenidos

**Objetivo**: Competir con Calm, Headspace y superar mediante sincronización grupal con IA.

**Diferenciador único**: Sistema de sincronización neuronal colectiva con narrativa IA adaptativa para equipos remotos.

## 🧘 MÓDULO 1: MEDITACIÓN

### Descripción
Sesiones guiadas de meditación con detección de estado mediante PPG y ajuste dinámico de contenido.

### Features Principales

1. **Tipos de Meditación**:
   - Mindfulness (atención plena)
   - Respiración (box breathing, 4-7-8)
   - Body scan (escaneo corporal)
   - Visualización guiada
   - Meditación transcendental
   - Loving-kindness (metta)

2. **Duración Configurable**:
   - Express: 3-5 min
   - Estándar: 10-15 min
   - Profunda: 20-30 min
   - Custom: usuario define

3. **Detección Biométrica**:
   - Monitor BPM en tiempo real
   - Calcular coherencia cardíaca
   - Objetivo: Alcanzar coherencia >70%
   - Feedback visual cuando se alcanza

4. **Audio Adaptativo**:
   - Voz guía (TTS personalizable)
   - Música ambiente (selección de 10+ tracks)
   - Binaurales Alpha (8-12Hz) para foco
   - Volumen auto-ajustable según fase

5. **Gamificación**:
   - Streak counter (días consecutivos)
   - Badges por hitos (7 días, 30 días, 100 sesiones)
   - Tiempo total meditado
   - "Coherencia promedio" score

6. **Visualización**:
   - Círculo de respiración animado
   - Onda de coherencia cardíaca en vivo
   - Partículas ambient que reaccionan a BPM

### UI/UX

```
[Header: Logo, Timer, Settings]

  - Círculo grande animado (respira)
  - BPM display central
  - Barra de coherencia debajo

  - Skip guide
  - Volumen música/voz
  - Terminar sesión

[Post-sesión]
```

## 😰 MÓDULO 2: REDUCCIÓN DE ESTRÉS

### Descripción
Sesiones cortas e intensivas para reducir estrés agudo mediante técnicas de respiración, coherencia cardíaca y estímulos EMDR.

### Features Principales

1. **Protocolos de Estrés**:
   - Quick Relief (3 min - emergencia)
   - Anxiety Reset (7 min)
   - Panic Recovery (10 min con EMDR)
   - Work Break (5 min - entre reuniones)
   - Pre-sleep Calm (15 min)

2. **Detección de Nivel de Estrés**:
   - Baseline BPM al inicio
   - Calcular HRV (RMSSD, SDNN)
   - Score de estrés (0-100):
     ```
     Estrés = 100 - HRV_coherence
     Bajo: <30
     Moderado: 30-60
     Alto: 60-85
     Severo: >85
     ```

3. **Técnicas Implementadas**:
   
   **A. Respiración Guiada**:
   - Box Breathing (4-4-4-4)
   - 4-7-8 Technique
   - Resonant Breathing (5.5 breaths/min)
   - Guía visual + conteo audio
   
   **B. Coherencia Cardíaca**:
   - Objetivo: Sincronizar respiración con BPM
   - Target: 6 respiraciones/min = coherencia óptima
   - Feedback inmediato visual
   
   **C. EMDR Auditivo** (opcional):
   - Estimulación bilateral L/R
   - Beat frequency 4Hz (Theta)
   - Para procesar trauma/ansiedad
   
   **D. Grounding 5-4-3-2-1**:
   - Guía interactiva
   - Input usuario (texto/voz)

4. **Visualización de Progreso**:
   - Gráfico BPM en tiempo real
   - Barra de "Calma" que sube al mejorar HRV
   - Confeti cuando alcanza objetivo

5. **Intervenciones IA**:
   - Si estrés no baja en 3 min → Cambiar técnica
   - Si aumenta → Sugerir pausa o contactar ayuda profesional
   - Mensajes de ánimo personalizados

## 😴 MÓDULO 3: SUEÑO PROFUNDO

### Descripción
Análisis de fases de sueño mediante audio (respiración) y PPG, con afirmaciones subliminales en sueño profundo e inducción de sueños lúcidos en REM.

### Features Principales

1. **Preparación para Dormir**:
   - Rutina pre-sueño guiada (15-30 min)
   - Técnicas de wind-down:
     - Progressive muscle relaxation
     - Visualización (lugar tranquilo)
     - Gratitude journaling (voice-to-text)
   - Música binaural Delta (0.5-4Hz)

2. **Monitoreo Durante el Sueño**:
   
   **Detección de Fases** (vía respiración + PPG):
   ```
   AWAKE: 15-25 breaths/min, BPM >65
   DROWSY: 12-18 breaths/min, BPM 60-70
   LIGHT_SLEEP: 10-15 breaths/min, BPM 55-65
   DEEP_SLEEP: 6-12 breaths/min, BPM 45-55 ← AFIRMACIONES
   REM: 12-20 breaths/min irregular, BPM 55-70 ← LUCID TRIGGERS
   ```
   
   **Validación Cruzada**:
   - Audio + PPG = Mayor confiabilidad
   - Mínimo 2 min en fase para confirmar

3. **Afirmaciones Subliminales**:
   
   **Cuándo**:
   - Solo en DEEP_SLEEP confirmado
   - Mínimo 5 min en fase antes de primera afirmación
   - Intervalo: 30-45 seg entre afirmaciones
   
   **Cómo**:
   - Volumen ultra-bajo (3-5% volumen sistema)
   - Pitch shift -200Hz (más subliminal)
   - Duración: 8-12 seg c/u
   - TTS con voz calma (usuario puede elegir)
   
   **Contenido**:
   - Categorías:
     - Sueño reparador
     - Auto-confianza
     - Procesamiento emocional
     - Creatividad
     - Salud física
   - Usuario puede crear custom o usar biblioteca (200+)
   
   **Ejemplos**:
   ```
   "Mi cuerpo se regenera completamente"
   "Despierto renovado y lleno de energía"
   "Suelto el estrés del día"
   "Mi mente procesa y aprende mientras duermo"
   ```

4. **Inducción de Sueños Lúcidos**:
   
   **Triggers en REM**:
   - Pulso de 40Hz (Gamma) muy sutil
   - Duración: 3-5 segundos
   - Intervalo: 5-10 min
   - Objetivo: Aumentar awareness sin despertar
   
   **Afirmaciones REM**:
   - "Estoy soñando"
   - "Reconozco este es un sueño"
   - "Tengo control de mis sueños"
   
   **Reality Checks** (pre-sueño):
   - App pregunta: "¿Estás soñando?" durante el día
   - Crear hábito de verificar realidad

5. **Despertar Inteligente**:
   - Alarma en fase Light/REM (no Deep)
   - Ventana de 30 min alrededor de hora objetivo
   - Despertar gradual con luz + sonido
   - Pregunta: "¿Recordás tus sueños?"

6. **Análisis Post-Sueño**:
   - Hipnograma (gráfico de fases)
   - Tiempo en cada fase
   - Número de despertares
   - Calidad de sueño score (0-100)
   - Comparación con noches previas

7. **Grabación de Sesión**:
   - Audio completo (opcional)
   - Detección de ronquidos
   - Detección de apnea (pausa >10s)
   - Alertar si apnea frecuente

## 🚀 MÓDULO 4: RENDIMIENTO / PEAK PERFORMANCE

### Descripción
Optimizar estados de flow, focus y energía para trabajo profundo, creatividad o ejercicio mediante sincronización de ondas cerebrales.

### Features Principales

1. **Modos de Rendimiento**:

   **A. Deep Work (Trabajo Profundo)**:
   - Duración: 25, 50, 90 min (Pomodoro)
   - Objetivo: Beta bajo (12-15Hz) → Flow
   - Binaurales: Beta + White noise
   - Sin interrupciones (modo avión sugerido)
   - Timer Pomodoro integrado
   
   **B. Creative Flow (Creatividad)**:
   - Duración: 30-60 min
   - Objetivo: Theta-Alpha (6-10Hz)
   - Binaurales: Theta waves
   - Prompts creativos cada 15 min (IA)
   - Captura de ideas (voice-to-text)
   
   **C. High Energy (Pre-ejercicio/Presentación)**:
   - Duración: 10-15 min
   - Objetivo: Beta alto (15-30Hz) + Gamma
   - Música energizante + Binaurales
   - Afirmaciones de confianza
   
   **D. Learning Mode (Estudio)**:
   - Duración: 45-120 min
   - Objetivo: Alpha (8-12Hz)
   - Binaurales + Música clásica (Mozart Effect)
   - Quiz breaks cada 30 min

2. **Detección de Estado**:
   - BPM baseline al inicio
   - Monitoring continuo HRV
   - Detectar "salida de flow":
     ```
     Flow = HRV coherence >60% + BPM estable
     Out of Flow = Coherence <40% por >2 min
     ```
   - Intervención si pierde focus

3. **Intervenciones Inteligentes**:
   - Si pierde flow → Micro-break (2 min respiración)
   - Si energía baja → Sugerir movimiento (30 saltos)
   - Si muy activado → Bajar binaurales
   - Mensajes motivacionales (IA personalizada)

4. **Integración con Calendario**:
   - Conectar Google Calendar / Outlook
   - Detectar bloques de "Deep Work"
   - Auto-iniciar sesión si usuario acepta
   - Silenciar notificaciones externas

5. **Métricas de Productividad**:
   - Tiempo en flow hoy/semana
   - Mejor hora del día (pico performance)
   - Correlación sueño ↔ rendimiento
   - Insights IA semanales

6. **Leaderboard (Opcional)**:
   - Competir con amigos
   - "Minutos de flow" esta semana
   - Badges: "Flow Master", "Deep Worker", etc.

## 👥 MÓDULO 5: SINCRONIZACIÓN GRUPAL

### Descripción
**El diferenciador clave de la plataforma**. Sistema de sincronización neuronal colectiva con narrativa IA adaptativa para equipos remotos.

### Features Principales

1. **Creación de Sesión Grupal**:

   **Roles**:
   - **Facilitador**: Crea sesión, controla fases, ve métricas detalladas
   - **Participante**: Solo ve su BPM y coherencia grupal
   - **Observador**: Solo mira (útil para managers)
   
   **Config Inicial**:
   - Nombre de sesión
   - Objetivo: [Brainstorming, Alineación, Resolución problemas, Team building]
   - Duración: 20, 30, 45, 60 min
   - Participantes: Hasta 8 gratis, hasta 50 en Pro, ilimitado en Enterprise
   - Contexto corporativo (opcional):
     - Upload docs (PDF, DOCX)
     - Objetivos del equipo
     - Jerga específica

2. **Fases de la Sesión**:

   **Estructura de 30 min típica**:
   ```
   1. LOBBY (0-2 min)
      - Participantes se unen
      - Test audio/video/PPG
      - Introducción facilitador
   
   2. CALIBRACIÓN (2-5 min)
      - Cada uno coloca dedo en cámara
      - Obtener baseline BPM individual
      - Objetivo: Todos >40% calidad señal
   
   3. SINCRONIZACIÓN (5-10 min)
      - Respiración guiada grupal
      - Binaurales Theta → Alpha (4→10Hz)
      - Objetivo: Coherencia grupal >70%
      - Visual: Círculos de cada persona sincronizándose
   
   4. NARRATIVA COMPARTIDA (10-20 min)
      - IA genera metáfora/historia basada en:
        * Objetivo de sesión
        * Contexto corporativo
        * Estado biométrico del grupo
      - Audio TTS reproducido a todos simultáneamente
      - Mantener Alpha (8-12Hz)
      - Objetivo: "Espacio mental compartido"
   
   5. CREATIVIDAD EXPLOSIVA (20-25 min)
      - Rampa a Gamma (40Hz)
      - IA genera provocaciones/preguntas:
        "¿Y si pensamos esto al revés?"
        "¿Qué haría [empresa inspiradora]?"
      - Participantes pueden escribir insights (anónimo)
   
   6. INTEGRACIÓN (25-30 min)
      - Bajar a Beta (12-18Hz)
      - Resumen de insights generados
      - Action items sugeridos por IA
      - Cierre
   ```

3. **Sincronización en Tiempo Real**:

   **WebSocket Events**:
   ```typescript
   // Cada participante envía cada 2 seg
   socket.emit('bio:update', {
     sessionId,
     userId,
     bpm: 72,
     hrv: 45.2,
     signalQuality: 85
   })
   
   // Servidor responde a todos
   socket.emit('group:metrics', {
     avgBPM: 68,
     groupCoherence: 73, // 0-100
     phase: 'ALPHA_NARRATIVE',
     outliers: [userId3], // Fuera de rango
     nextPhaseIn: 180 // segundos
   })
   ```
   
   **Agregación Biométrica**:
   ```typescript
   function calculateGroupCoherence(participants) {
     const bpms = participants.map(p => p.bpm).filter(b => b > 0);
     const mean = average(bpms);
     const stdDev = standardDeviation(bpms);
     
     // Coherencia = qué tan cerca están todos
     const coherence = Math.max(0, 100 - (stdDev / mean) * 100);
     return coherence;
   }
   ```

4. **Sistema de Narrativa IA**:

   **Prompt Structure** (Anthropic Claude):
   ```
   Eres un facilitador de experiencias grupales de sincronización neuronal.
   
   CONTEXTO:
   - Equipo: {teamName}
   - Objetivo sesión: {goal}
   - Contexto corporativo: {corporateContext}
   - Fase actual: {phase}
   - Métricas bio: BPM promedio {avgBPM}, Coherencia {coherence}%
   
   TAREA:
   Genera una narrativa de 2 minutos para la fase {phase}.
   
   Si phase === ALPHA_NARRATIVE:
     - Tono: Calmado, visual, metafórico
     - Objetivo: Crear "espacio mental compartido"
     - Conectar el objetivo con una imagen/metáfora universal
     - Incluir nombres de participantes sutilmente
     - Ejemplo: "Imaginen que son un río... cada uno es una corriente..."
   
   Si phase === GAMMA_PROVOCATION:
     - Tono: Energético, desafiante, provocador
     - Genera 3 preguntas que desafíen assumptions
     - Perspectivas no obvias
     - Conexiones inesperadas con el contexto corporativo
   ```

### Tecnología Específica

```typescript
// Server-side (Node.js)
import { Server } from 'socket.io';
import Anthropic from '@anthropic-ai/sdk';

interface Participant {
  id: string;
  name: string;
  bpm: number;
  hrv: number;
  lastUpdate: number;
  signalQuality: number;
}

interface Session {
  id: string;
  facilitatorId: string;
  participants: Map<string, Participant>;
  phase: 'LOBBY' | 'CALIBRATING' | 'SYNCING' | 'ALPHA_NARRATIVE' | 'GAMMA_PROVOCATION' | 'INTEGRATION';
  startTime: number;
  config: SessionConfig;
  insights: string[];
}

class GroupSyncManager {
  private sessions: Map<string, Session> = new Map();
  private io: Server;
  private anthropic: Anthropic;
  
  constructor(io: Server) {
    this.io = io;
    this.anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });
  }
  
  async handleBioUpdate(sessionId: string, userId: string, data: { bpm: number, hrv: number }) {
    const session = this.sessions.get(sessionId);
    if (!session) return;
    
    // Update participant
    const participant = session.participants.get(userId);
    if (participant) {
      participant.bpm = data.bpm;
      participant.hrv = data.hrv;
      participant.lastUpdate = Date.now();
    }
    
    // Calculate group metrics
    const groupMetrics = this.calculateGroupMetrics(session);
    
    // Broadcast to all in room
    this.io.to(sessionId).emit('group:metrics', groupMetrics);
    
    // Check if should transition phase
    if (this.shouldTransitionPhase(session, groupMetrics)) {
      await this.transitionPhase(session);
    }
  }
  
  calculateGroupMetrics(session: Session) {
    const participants = Array.from(session.participants.values())
      .filter(p => p.bpm > 0 && Date.now() - p.lastUpdate < 5000); // Active in last 5s
    
    if (participants.length === 0) {
      return { avgBPM: 0, groupCoherence: 0, activeCount: 0 };
    }
    
    const bpms = participants.map(p => p.bpm);
    const avgBPM = bpms.reduce((a, b) => a + b, 0) / bpms.length;
    
    // Calculate coherence (inverse of coefficient of variation)
    const stdDev = this.calculateStdDev(bpms);
    const cv = stdDev / avgBPM;
    const groupCoherence = Math.max(0, Math.min(100, (1 - cv) * 100));
    
    return {
      avgBPM: Math.round(avgBPM),
      groupCoherence: Math.round(groupCoherence),
      activeCount: participants.length,
      outliers: this.detectOutliers(participants, avgBPM, stdDev)
    };
  }
  
  async transitionPhase(session: Session) {
    const nextPhase = this.getNextPhase(session.phase);
    session.phase = nextPhase;
    
    // Generate AI narrative if needed
    if (nextPhase === 'ALPHA_NARRATIVE' || nextPhase === 'GAMMA_PROVOCATION') {
      const narrative = await this.generateNarrative(session);
      const audioURL = await this.textToSpeech(narrative);
      
      this.io.to(session.id).emit('phase:change', {
        phase: nextPhase,
        narrative,
        audioURL
      });
    } else {
      this.io.to(session.id).emit('phase:change', { phase: nextPhase });
    }
  }
  
  async generateNarrative(session: Session): Promise<string> {
    const prompt = this.buildNarrativePrompt(session);
    
    const message = await this.anthropic.messages.create({
      model: 'claude-3-5-sonnet-20241022',
      max_tokens: 1000,
      messages: [{
        role: 'user',
        content: prompt
      }]
    });
    
    return message.content[0].type === 'text' ? message.content[0].text : '';
  }
}
```

```typescript
callbacks: {
  async jwt({ token, user }) {
    if (user) {
      token.userId = user.id;
      token.plan = user.plan;
    }
    return token;
  },
  async session({ session, token }) {
    session.userId = token.userId;
    session.plan = token.plan;
    return session;
  }
}
```

## Base de Datos (PostgreSQL)

```sql
-- Core tables

CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email VARCHAR(255) UNIQUE NOT NULL,
  name VARCHAR(255),
  avatar_url TEXT,
  plan VARCHAR(50) DEFAULT 'FREE',
  stripe_customer_id VARCHAR(255),
  subscription_id VARCHAR(255),
  subscription_status VARCHAR(50),
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE sessions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id) ON DELETE CASCADE,
  module VARCHAR(50) NOT NULL, -- meditation, stress, sleep, performance, group_sync
  duration_seconds INT,
  started_at TIMESTAMP NOT NULL,
  completed_at TIMESTAMP,
  rating INT CHECK (rating >= 1 AND rating <= 5),
  metadata JSONB, -- Flexible data per module
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE biometric_data (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  session_id UUID REFERENCES sessions(id) ON DELETE CASCADE,
  timestamp TIMESTAMP NOT NULL,
  bpm INT,
  hrv_rmssd FLOAT,
  hrv_sdnn FLOAT,
  coherence FLOAT,
  signal_quality INT,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_biometric_session ON biometric_data(session_id);
CREATE INDEX idx_biometric_timestamp ON biometric_data(timestamp);

CREATE TABLE sleep_sessions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  session_id UUID REFERENCES sessions(id) ON DELETE CASCADE,
  phases JSONB, -- [{phase, startTime, duration}]
  affirmations_played INT DEFAULT 0,
  lucid_triggers INT DEFAULT 0,
  quality_score INT,
  recording_url TEXT,
  dream_log TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE group_sessions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  facilitator_id UUID REFERENCES users(id),
  join_code VARCHAR(10) UNIQUE,
  goal TEXT,
  status VARCHAR(50), -- lobby, active, completed
  context_docs JSONB,
  started_at TIMESTAMP,
  completed_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE group_participants (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  group_session_id UUID REFERENCES group_sessions(id) ON DELETE CASCADE,
  user_id UUID REFERENCES users(id),
  role VARCHAR(50), -- facilitator, participant, observer
  joined_at TIMESTAMP DEFAULT NOW(),
  left_at TIMESTAMP
);

CREATE TABLE ai_narratives (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  group_session_id UUID REFERENCES group_sessions(id),
  phase VARCHAR(50),
  prompt TEXT,
  generated_text TEXT,
  audio_url TEXT,
  effectiveness_rating INT,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE content (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  type VARCHAR(50), -- article, ebook, audio, video, course
  title VARCHAR(500) NOT NULL,
  slug VARCHAR(500) UNIQUE NOT NULL,
  description TEXT,
  content TEXT,
  thumbnail_url TEXT,
  file_url TEXT,
  duration_seconds INT,
  category VARCHAR(100),
  tags JSONB,
  access_level VARCHAR(50) DEFAULT 'FREE', -- FREE, PRO, PREMIUM
  rating FLOAT,
  views INT DEFAULT 0,
  published BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_content_type ON content(type);
CREATE INDEX idx_content_category ON content(category);
CREATE INDEX idx_content_slug ON content(slug);

CREATE TABLE user_favorites (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id) ON DELETE CASCADE,
  content_id UUID REFERENCES content(id) ON DELETE CASCADE,
  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(user_id, content_id)
);

CREATE TABLE subscriptions_log (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  event VARCHAR(100), -- created, updated, cancelled
  plan VARCHAR(50),
  amount DECIMAL(10,2),
  stripe_event_id VARCHAR(255),
  created_at TIMESTAMP DEFAULT NOW()
);
```
